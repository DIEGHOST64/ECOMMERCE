## Nota:
Esta refactorización es compleja y requiere mover ~60 archivos y actualizar ~200 imports.
Por limitaciones de las herramientas disponibles, se recomienda hacer esta refactorización manualmente en VS Code.

El proceso automatizado tendría alto riesgo de errores. VS Code tiene herramientas de refactoring que:
1. Actualizan imports automáticamente
2. Detectan referencias rotas
3. Permiten preview antes de aplicar cambios

## Alternativa Recomendada:
Seguir la guía paso a paso proporcionada usando las herramientas de refactoring de VS Code.
